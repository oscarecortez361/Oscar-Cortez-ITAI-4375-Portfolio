# Assignment 9: Designing a Reinforcement Learning Scenario in Healthcare

This folder contains my Assignment 9 submission for ITAI 4375 – AI in Healthcare.

## Overview
This assignment explored a reinforcement learning scenario in healthcare using insulin dosing for hospitalized patients with diabetes as an example of sequential decision-making.

## What I learned
I learned how reinforcement learning can adapt decisions over time based on patient data, while also requiring strong safety checks, testing, and clinical oversight.

This folder contains my Assignment 4 submission for ITAI 4375 – AI in Healthcare.

## Overview
This assignment evaluated an AI-based diagnostic test for skin cancer and discussed how measures such as sensitivity and specificity are used to judge test performance.

## What I learned
I learned how AI diagnostic tools can support early detection while still requiring clinical judgment and careful evaluation of accuracy and limitations.

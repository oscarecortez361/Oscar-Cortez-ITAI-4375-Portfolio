# Assignment 8: Critiquing a Natural Language Processing Study in Healthcare

This folder contains my Assignment 8 submission for ITAI 4375 – AI in Healthcare.

## Overview
This assignment critiqued a natural language processing study in healthcare and discussed how NLP can help analyze unstructured clinical notes while also raising concerns about bias and accuracy.

## What I learned
I learned how NLP can improve efficiency and decision-making in healthcare, but also why bias, fairness, and human oversight matter when these systems are used in clinical settings.

# Midterm: AI in Healthcare Glossary

This folder contains my midterm glossary project for ITAI 4375 – AI in Healthcare.

## Overview
This project defined key artificial intelligence, machine learning, deep learning, and healthcare evaluation terms used in modern healthcare systems.

## What I learned
I learned how concepts such as sensitivity, specificity, AUROC, bias, explainability, and clinical decision support connect directly to safe and effective healthcare AI.
